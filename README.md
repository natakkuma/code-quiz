# Code Quiz

<h2>Purpose</h2>
An online game that quizzes you on coding knowledge. All questions must be answered before the timer reaches zero. The user can save their initials and score as well as review the high score history. 

<h2>Built With</h2>
<ul>
<li>JavaScript
<li>CSS
<li>HTML
</ul>
